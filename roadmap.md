# EXECUTION ROADMAP: AI-Powered Vernacular Exam Prep Platform

**Total Horizon:** 12 months
**Goal:** Reach undeniable *outcome-based proof* before attempting scale or fundraising.

---

## PHASE 0 — Founder Readiness (Week 0–2)
**Objective:** Avoid false starts.
- [ ] Pick **ONE exam + ONE language** (e.g., SSC-GD (Hindi) or State PSC (Tamil))
- [ ] Define *success outcome* (e.g., “80% learners improve mock scores by 30% in 90 days”)
- [ ] Decide execution mode: Solo + contractors OR 1 technical + 1 academic co-founder
- [ ] **Deliverables:**
    - One-page problem statement
    - Defined learner persona
    - "What we will NOT build" list

## PHASE 1 — Problem Validation (Weeks 3–6)
**Objective:** Validate pain before building.
- [ ] Interview **50–75 aspirants** (Telegram, Coaching centers, YouTube)
- [ ] Identify top 5 failure patterns & willingness-to-pay
- [ ] **Deliverables:**
    - Validation report (Patterns, Friction points)
    - Go/No-Go decision based on pain clarity

## PHASE 2 — MVP Build (Weeks 7–12)
**Objective:** Build *minimum system*, not platform.
- [ ] **Core Features:**
    1. Diagnostic Test Engine
    2. AI Study Plan Generator
    3. Content delivery (video + PDF)
    4. Mentor check-in workflow
    5. Weekly mock tests
    6. Simple progress dashboard
- [ ] **Tech Stack (Lean):**
    - Frontend: React / Next.js
    - Backend: Firebase / Supabase
    - AI: OpenAI / open-source LLMs
    - Video: Unlisted YouTube / Vimeo
    - Communication: WhatsApp + Telegram
- [ ] **Deliverables:**
    - Working MVP
    - 30–50 users onboarded manually

## PHASE 3 — First Cohort Launch (Months 4–5)
**Objective:** Prove outcomes, not scale.
- [ ] Run **30–60 day cohort**
- [ ] Daily AI study plans & Weekly mentor reviews
- [ ] Charge early (₹1,000–₹2,000)
- [ ] **Deliverables:**
    - 10–15 success stories
    - Data-backed improvement graphs
    - Refined curriculum

## PHASE 4 — Trust Layer & Product Refinement (Months 6–7)
**Objective:** Turn results into credibility.
- [ ] Publish outcome dashboards & "Score Improvement Guarantee"
- [ ] Standardize Mentor playbooks
- [ ] **Deliverables:**
    - Trust page
    - Repeated cohort SOP

## PHASE 5 — Controlled Growth (Months 8–10)
**Objective:** Grow without breaking trust.
- [ ] Expansion: Same exam -> new state OR Same language -> adjacent exam
- [ ] Channels: YouTube, Telegram, Referrals
- [ ] **Targets:** 300–500 active learners, CAC < ₹1,000

## PHASE 6 — Scale Readiness (Months 11–12)
**Objective:** Become fundable.
- [ ] Reduce mentor-to-student ratio via AI
- [ ] Prepare pitch metrics (LTV, CAC, Outcome consistency)
- [ ] **Deliverables:** Investor-ready metrics
