# Mentor Operations Manual & Playbook

**trust backbone of the platform.**

## 1. Mentor Role Definition
**Mentors are:**
- Accountability guides
- Performance interpreters
- Motivation stabilizers
**Mentors are NOT:**
- Teachers / Influencers / Content creators

## 2. Ideal Mentor Profile
- **Mandatory**: Cleared same/higher exam OR 2+ years guiding success.
- **Traits**: Strong vernacular communication, Calm/Structured.
- **Red Flags**: Over-motivational speakers, "Concept teachers".

## 3. Hiring Funnel
1. **Screening**: Resume + Exam proof + Short vernacular response.
2. **Case Simulation**: "Student score dropped after 10 days study. What do you do?" (Test Logic/Empathy).
3. **Live Mock Session (20 min)**: Review fake dashboard, explain next steps.
4. **Trial Period (Paid)**: 2 weeks, 5-10 students.

## 4. Mentor Playbook (SOP)

### Weekly Workflow
**1. Before Session**
- Review dashboard: Identify missed days, score drops, weak topics.
**2. During Session (15-20 min)**
- Review last week.
- Identify ONE core issue.
- Adjust workload (not content).
- Set next week target.
- Confirm commitment.
**3. After Session**
- Log notes.
- Flag risks.
- Rate engagement.

### Rules of Engagement
| Allowed | Forbidden |
| :--- | :--- |
| Adjust intensity | Teach concepts |
| Recommend revision | Change syllabus |
| Escalate disengagement | Exam predictions |
| Motivate calmly | "Guarantees" |
